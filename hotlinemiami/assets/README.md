# Game Assets

Place your Hotline Miami Android game files here.

## Required Files

From the Android APK's `assets/` folder, copy:
- `game.droid` (required)
- All `.ogg` audio files (required for music/sounds)

## How to Extract

1. Get the Hotline Miami Android APK
2. Use 7-Zip or similar to extract the APK
3. Copy files from the `assets/` folder to this directory
