## Notes

Thanks to Dennaton Games and Devolver Digital for creating Hotline Miami!

This port requires game files from the **Android version** of Hotline Miami.

## Installation

1. Download `hotlinemiami.zip`
2. Place the ZIP file in your device's `ports/autoinstall/` folder
3. Launch Portmaster - it will automatically install the port
4. Get your Hotline Miami Android APK
5. Rename the APK to `game.port`
6. Place `game.port` in the `ports/hotlinemiami/` folder
7. Launch the game via Portmaster

## Controls

| Button | Action |
|--------|--------|
| D-Pad / Left Analog | Movement |
| Right Analog | Aim |
| A | Attack |
| B | Use / Pickup |
| X | Lock Aim |
| Y | Restart Level |
| L1 | Previous Weapon |
| R2 | Throw Weapon |
| Start | Pause |
| Select | Show Score |
